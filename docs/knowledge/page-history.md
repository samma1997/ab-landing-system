## [MI-V2] — Missione Immobiliare (v2)
**Built:** 2026-03-19 / Updated: 2026-03-20
**Status:** live
**Slug:** /missione-immobiliare-v2
**Route type:** hardcoded (Next.js page file at `src/app/missione-immobiliare-v2/page.tsx`)
**Template used:** workshop
**Deploy target:** Vercel

### Blocks Used (in order)
1. `hero` — HeroAnimated with dark overlay + orange gradient
2. `social-proof` — SocialProofAnimated (stats + media logos)
3. `benefits` — BenefitsAnimated (4 serate program)
4. `speaker` — SpeakerSection (Alfio Bardolla bio)
5. `event-details` — EventDetailsSection (dates, venue, format)
6. `testimonials` — TestimonialSection (student results)
7. `faq` — FAQSection (6 questions)
8. `countdown` — CountdownTimer (to event start date)
9. `hubspot-form` — HubSpotFormSection (registration)
10. `footer` — FooterSection (standard ABTG)
+ Conversion overlays: StickyCtaBar, CountdownBanner, SocialProofTicker

### Input
- PDF: Content provided describing 4-evening free workshop on real estate investing
- Event: 4 serate, 27-30 Aprile 2026, location TBD (online/Milan)
- Images provided: No — all placeholders used
- HubSpot Form ID: not set (placeholder config)
- Facebook Pixel ID: not set

### Design Decisions Made

**Color scheme:** Full ABTG dark theme (#0F0F1A background, #D76E11 orange accents). Chosen over a lighter theme because financial education authority brands better dark.

**Animation approach:** GSAP ScrollTrigger for all section entrances. Lenis smooth scroll enabled. Particle effects disabled on mobile for performance.

**Block ordering rationale:** Followed the Conversion Playbook blueprint for "free event" page type:
- Hero first (obvious)
- Social proof immediately after hero (skepticism reduction before pitch)
- Benefits before speaker (tell what they get before who delivers it)
- Speaker after benefits (now they care who delivers)
- Testimonials as social proof of the outcome
- FAQ just above the form (handle objections right before conversion point)
- Countdown + form together (urgency + action in same viewport)

**What was generated by Claude (not in PDF):**
- Meta title and description
- Page slug
- Countdown end date (set to 2026-04-27T09:00:00)
- OG image placeholder
- FAQ answers expanded from brief PDF notes

### Known Issues

- Images: All images are placeholder (`/images/placeholder-*.jpg`). When real photos are available, add to bank and update page.
- HubSpot: Form renders but form ID is placeholder — update before going live with real campaign.
- Facebook Pixel: Not configured — add when campaign is active.
- Footer URLs: Privacy policy URL uses generic `#` — update with real ABTG legal page URL.

### Conversions / Feedback

- Marketing team review: pending
- Conversion data: not yet available (page live but campaign not started)

### What to Replicate on Next Workshop Page

- The sticky CTA bar pattern works well — keep it for all event pages
- Benefits section as animated grid (4 cards) reads cleanly
- SocialProofTicker at top of page builds immediate credibility
- Dark background with orange accents is visually consistent with ABTG brand

### What to Change Next Time

- Need real speaker photo in bank before building — placeholder hurts credibility
- FAQ section needs more questions for paid events (6 is minimum, 8-10 better)
- Event details section could benefit from a map embed for in-person events
- The form section should have a headline like "Prenota il tuo posto" not just the form
